# SkillProof v1.0.0 Release Notes

Release date: 2026-03-08

## Highlights

- Official 1.0 launch with new product naming:
  - Brand: `SkillProof`
  - npm package: `skillproof`
  - Primary CLI command: `skillproof`
  - Claude plugin slash commands: `/skillproof-*`
- Reduced LLM token cost in skill inference pipeline.
- Strengthened bundle verification security model.
- Documentation synchronized with current behavior and security guarantees.

## Breaking/Behavior Changes

- Primary command moved from `skillproof` to `skillproof`.
- Legacy slash command files `commands/skillproof-*.md` were removed.
- New slash command files are `commands/skillproof-*.md`.

## Compatibility

- Internal data directory remains `.skillproof/` for backward compatibility.
- Manifest path remains `.skillproof/resume-manifest.json`.

## Token Cost Optimizations Included

- Skill grouping to deduplicate overlapping file context.
- Cache key strengthened with explicit model dimension.
- Shared `LLM_MODEL` constant to avoid cache/model mismatch.
- Global review budget and pre-review cost preview.
- `--dry-run` flow for zero-cost estimate before LLM calls.

## Security Improvements Included

- Verification uses signed `manifest.file_hashes` for integrity checks.
- `verification.json` treated as informational only.
- Zip Slip protection checks archive entries before extraction.
- `doctor` enforces `unzip` and `zipinfo` as required for `verify`.
